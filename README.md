# Professional Developer Portfolio

A responsive, modern, and interactive portfolio website showcasing my projects, technical skills, and experience as a Frontend and Full-Stack Developer.

🚀 **Live Link:** [portfolio-wakeupsaket.vercel.app](https://portfolio-wakeupsaket.vercel.app)

---

## 🛠️ Tech Stack & Skills

*   **Frontend:** HTML5, CSS3, JavaScript (ES6+), Responsive Web Design
*   **Aesthetics:** Modern typography, custom layout grids, clean dark mode transitions
*   **Deployment:** Vercel

---

## 🌟 Key Features

*   **Responsive Layout:** Fully optimized for desktop, tablet, and mobile screens.
*   **Interactive UI:** Smooth transitions, micro-interactions, and engaging hover effects.
*   **Project Showcase:** Interactive cards displaying detailed project info, technologies used, and live demo links.
*   **Contact Integration:** Seamless email and social media channel integration.

---

## 📂 Project Structure

```
Portfolio/
├── assets/          # Images, logos, and custom icons
├── index.html       # Structural layout & SEO-optimized markup
├── style.css        # Custom CSS styling (gradients, animations, grid/flexbox)
├── script.js        # Dynamic interactive elements & scroll behaviors
├── .gitignore       # Untracked files list
└── vercel.json      # Vercel deployment configuration
```

## 🚀 How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Saket-Chawla/Portfolio.git
    ```
2.  **Open in your browser:**
    Simply open the `index.html` file in any modern web browser or run it using a VS Code Live Server extension.
